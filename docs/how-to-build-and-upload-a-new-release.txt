How to build a release
======================

Update the version number:
  
    vim trashcli/trash.py
    version="$(python setup.py --version)"

Run all tests:

    pytest

Create the tarball:
  
    python setup.py sdist

Register and upload:

    python setup.py sdist upload

Now you can tag the repo status:

    git tag "${version:?}"
